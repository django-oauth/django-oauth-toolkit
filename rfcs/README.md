# RFC and Spec Corpus

This directory contains plain-text standards documents used by this repository.

## Scope

- IETF OAuth and related RFCs referenced by code/docs/tests.
- OpenID Connect specifications used by implemented OIDC features.
- Security BCP is included as `rfc9700.txt`.
